## How to Use

1. Start with lesson `01-classes-objects`
2. Read the `README.md` in each lesson
3. Run the examples in `src/Main.java`
4. Apply what you learned in `school/` folder
5. Complete the exercises
6. Move to the next lesson

## Final Project Features

The School Management System will include:

- **Models:** Person, Student, Teacher, Course, Grade
- **Features:**
  - Add/remove students and teachers
  - Create courses
  - Enroll students in courses
  - Assign grades
  - Calculate GPA
  - Generate reports
- **OOP Concepts Used:**
  - Inheritance hierarchy
  - Encapsulation with validation
  - Polymorphism for different person types
  - Interfaces for shared behaviors
  - Enums for grade levels
  - Singleton for system management
